Projet de développement d'un jeu de Tower Defense, réalisé dans le cadre du cours LO21 dispensé à l'UTC.
Auteurs : Victor Hatinguais, Louis Pontoise

code : contient les fichiers sources .cpp, .h et .pro (Projet Qt)
doxygen : contient une documentation doxygen très complète du projet
data : contient les fichiers de ressource qui doivent être placés au même niveau que l'exécutable, éventuellement accompagnés des .dll requises (Windows)
	- map.txt et waves.txt contiennent les fichiers de génération du niveau
	- *.jpg sont les images utilisées par le jeu
